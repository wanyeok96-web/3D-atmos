# 대기대순환 3D — 수업 시연용 학습자료

지구의 바람 흐름이 비와 기후를 만드는 과정을 3D로 보여주는 **개념 이해용 시연 자료**입니다.
퀴즈·점수·로그인·저장 기능이 없는 순수 설명 도구로, 교사의 수업 시연과 학생의 자유 탐색에 모두 쓸 수 있습니다.

## 실행 방법

1. 폴더 구조를 그대로 유지한 채 **index.html을 더블클릭**하면 바로 실행됩니다. (설치·서버 불필요)
2. 크롬(Chrome) 또는 엣지(Edge) 브라우저 권장.

```
대기대순환3D/
├── index.html      ← 이 파일을 더블클릭
├── style.css
├── script.js
└── libs/
    └── three.min.js   (3D 라이브러리 — 반드시 함께 있어야 함)
```

> 인터넷이 연결되어 있으면 자연색(위성사진 계열) 지구 텍스처를 불러오고,
> **오프라인이면 내장된 교육용 지구본으로 자동 전환**됩니다. 기능은 동일하게 동작합니다.

## 주요 기능

- **보기 전환** — 지구 전체 보기(3D 지구본) ↔ 공기 흐름 단면 보기(남극~북극 단면)
- **이해 순서 7단계** — 단계를 누르면 화면·표시 내용이 자동 설정되고, 설명과 "생각해 볼 질문"이 함께 나옵니다.
  1. 적도에서는 왜 공기가 올라갈까?
  2. 올라간 공기는 어디로 갈까?
  3. 30° 부근에는 왜 사막이 많을까?
  4. 지구에는 공기의 큰 순환이 3개 있어요 (해들리·페렐·극)
  5. 땅 가까이에서는 바람이 어떻게 불까? (전향력 ON/OFF 비교)
  6. 올라가는 곳은 비, 내려오는 곳은 건조
  7. 계절에 따라 바람 띠가 움직여요
- **표시할 내용(레이어)** — 공기의 큰 순환 / 오르내리는 공기 띠(기압대) / 땅 가까이 부는 바람 / 바람 휘어짐(전향력) / 비 많은 곳·건조한 곳 / 위도·경도선
- **계절 슬라이더** — 1월 ↔ 7월로 움직이면 기압대·바람대가 남북으로 이동 (단면 보기에는 고정 위도 눈금이 있어 이동을 비교하기 쉬움)
- **위도대 클릭 카드** — 색깔 띠를 클릭하면 「공기의 움직임 / 비와 건조 / 대표 기후·지역 / 한 줄 정리」 4단 구성 카드가 나타납니다. 쉬운 말 우선, 교과 용어(적도 저압대 등) 병기.
- **조작** — 드래그로 지구 회전, 휠·두 손가락으로 확대/축소.

## 수업 활용 팁

- 처음에는 1단계부터 「다음 →」으로 순서대로 진행하며 흐름을 잡아 주세요.
- 5단계에서 **바람 휘어짐을 껐다 켜며** 전향력의 효과를 비교하면 반응이 좋습니다.
- 7단계에서 계절 슬라이더를 움직이며 사바나·지중해성 기후의 우기/건기와 연결해 보세요.
