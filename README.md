# 대기대순환 3D — 수업 시연용 학습자료

지구의 바람 흐름이 비와 기후를 만드는 과정을 3D로 보여주는 **개념 이해용 시연 자료**입니다.
퀴즈·점수·로그인·저장 기능이 없는 순수 설명 도구로, 교사의 수업 시연과 학생의 자유 탐색에 모두 쓸 수 있습니다.

## 실행 방법

1. 폴더 구조를 그대로 유지한 채 **index.html을 더블클릭**하면 바로 실행됩니다. (설치·서버 불필요)
2. 크롬(Chrome) 또는 엣지(Edge) 브라우저 권장.

```
대기대순환3D/
├── index.html      ← 이 파일을 더블클릭
├── style.css
├── script.js
└── libs/
    └── three.min.js   (3D 라이브러리 — 반드시 함께 있어야 함)
```

> 인터넷이 연결되어 있으면 자연색(위성사진 계열) 지구 텍스처를 불러오고,
> **오프라인이면 내장된 교육용 지구본으로 자동 전환**됩니다. 기능은 동일하게 동작합니다.

## 주요 기능

- **보기 전환** — 지구 전체 보기(3D 지구본, 기본) ↔ 공기 흐름 단면 보기(남극~북극 단면, 보조)
- **이해 순서 7단계** — 지구본을 기준으로 한 수업 시나리오. 단계를 누르면 화면·표시 내용이 자동 설정되고, 설명과 "생각해 볼 질문"이 함께 나옵니다.
  1. 대기대순환이란? — 위도별 일사량과 열적 불균형 (태양·평행 광선 애니메이션)
  2. 저위도 지역의 공기 흐름 (적도 상승 → 30° 하강)
  3. 고위도 지역의 공기 흐름 (극 하강 → 60° 상승)
  4. 대기대순환 시스템의 형성 — 3개의 순환 고리와 저기압 L / 고기압 H
  5. 강수 — 저기압은 비, 고기압은 맑음
  6. 바람 — 무역풍·편서풍·극동풍 (바람 휘어짐 ON/OFF 비교)
  7. 결론 — 대기대순환과 위도별 기후
- **표시할 내용(레이어)** — 햇빛과 기온(일사) / 공기의 큰 순환 / 오르내리는 공기 띠(기압대) / 땅 가까이 부는 바람 / 바람 휘어짐(전향력) / 비 많은 곳·건조한 곳 / 위도·경도선
- **위도대 클릭 카드** — 색깔 띠를 클릭하면 「공기의 움직임 / 비와 건조 / 대표 기후·지역 / 한 줄 정리」 4단 구성 카드가 나타납니다. 쉬운 말 우선, 교과 용어(적도 저압대 등) 병기.
- **조작** — 드래그로 지구 회전, 휠·두 손가락으로 확대/축소.

### 이번 업데이트 (피드백 반영)

- **보기 유지** — 공기 흐름 단면을 보는 중에 「다음 →」을 누르면 지구본으로 되돌아가지 않고 **단면인 채로 다음 단계**가 이어집니다. (1단계 일사 애니메이션만 지구본 전용이라 자동 전환)
- **3D 태양** — 태양이 평면 아이콘 대신 **표면 무늬가 있는 발광 구 + 코로나 + 글로우**로 그려지고 천천히 자전합니다. 1단계(햇빛과 기온)에서는 **지구보다 크게, 화면 오른쪽 가장자리에 절반쯤 걸치도록 고정**되어 지구를 돌려도 자리가 변하지 않습니다. 화면 비율·확대 정도에 맞춰 노출량이 자동 조절되어 지구를 가리지 않으며, 평행 광선과 입사 면적 표시도 항상 태양 쪽을 향합니다. (지구를 좌우로 돌리면 햇빛을 받는 면이 바뀌는 것을 관찰할 수 있어요.)
- **위도별 층후 표현** — 순환 고리가 해들리(저위도) → 페렐 → 극(고위도) 순으로 점점 **얇고 낮게** 그려집니다. 공기층(대류권)이 적도에서 두껍고 극으로 갈수록 얇아지는 것을 반영했으며, 단면 보기에는 공기층 높이를 나타내는 점선 안내선이 함께 표시됩니다.
- **모바일 하단 시트** — 휴대전화에서는 좌측 패널 대신 화면 아래에 **축소 바(현재 단계 + ← →)**가 나타나 지구를 가리지 않습니다. 바를 **위로 쓸어올리면** 패널이 확장되고, 확장 상태나 축소 바에서 **좌우로 쓸면** 단계가 넘어갑니다. 단계를 선택하면 시트가 자동으로 접혀 화면이 바로 보입니다.
- **3D 입체감 강화**
  - 순환 고리·바람 화살표가 빛을 받아 **볼륨감 있는 음영·하이라이트**로 표현됩니다.
  - 지구 가장자리에 **대기 산란(림 라이트)** 효과, 바다에 은은한 광택이 추가되었습니다.
  - 지구 위에 천천히 흐르는 **구름층과 표면 그림자**가 깊이감을 더합니다. (학습 레이어가 켜지면 자동으로 옅어져 내용을 가리지 않음)
  - 단계 전환 시 카메라가 **부드럽게 이동·확대/축소**되며, 드래그·휠을 시작하면 연출이 즉시 멈춰 조작을 방해하지 않습니다.

> 계절에 따른 기압대·바람대의 이동(8단계)은 추후 추가 예정입니다. (계절 슬라이더는 현재 숨김 상태)

## 수업 활용 팁

- 처음에는 1단계부터 「다음 →」으로 순서대로 진행하며 흐름을 잡아 주세요.
- 1단계의 일사 애니메이션으로 "왜 적도는 덥고 극지방은 추운가"를 먼저 짚으면 이후 단계가 자연스럽게 이어집니다.
- 6단계에서 **바람 휘어짐을 껐다 켜며** 전향력의 효과를 비교하면 반응이 좋습니다.
- 7단계에서 색깔 띠를 클릭해 위도대별 기후(열대 우림·사막·서안 해양성 등)와 연결해 보세요.
